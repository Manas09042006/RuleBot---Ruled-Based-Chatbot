RuleBot 🤖

RuleBot is a modern rule-based chatbot built using Python, CustomTkinter, Regular Expressions (Regex), and Object-Oriented Programming (OOP). It provides predefined responses based on user input by matching keywords and patterns with regular expressions.

The application features a modern desktop interface with a dark theme, interactive chat window, responsive controls, hover effects, timestamps, and an intuitive user experience. Unlike AI-powered chatbots, RuleBot follows predefined conversational rules, making it lightweight, fast, and ideal for learning chatbot fundamentals.

🚀 How It Works
The user enters a message.
The chatbot preprocesses the input.
Regular Expressions (Regex) compare the input against predefined patterns.
If a pattern matches, RuleBot returns the corresponding response.
If no pattern matches, a default response is displayed.
🎯 Purpose

This project demonstrates the implementation of a rule-based chatbot using Python without machine learning or artificial intelligence. It is suitable for beginners who want to understand chatbot architecture, GUI development, regex pattern matching, and object-oriented programming concepts.
