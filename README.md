# 🤖 RuleBot - Rule-Based Chatbot

A modern **Rule-Based Chatbot** built using **Python**, **CustomTkinter**, **Regular Expressions (Regex)**, and **Object-Oriented Programming (OOP)**.

RuleBot provides intelligent predefined responses by matching user input with regular expression patterns. It features a modern desktop GUI with a dark theme, responsive controls, timestamps, hover effects, and a clean user interface.

# ⚙️ Installation


### 2. Navigate to the Project

```bash

```

### 3. Install Dependencies

```bash

```

### 4. Run the Project

```bash

```

# 🚀 How It Works

1. The user enters a message.
2. RuleBot preprocesses the input.
3. Regex patterns are matched against the message.
4. If a pattern matches, the corresponding predefined response is returned.
5. Otherwise, a default response is displayed.
